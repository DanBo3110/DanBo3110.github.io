08 - 内容选择器

Oct 23, 2019点击

  1. :empty
    作用:找到既没有文本内容也没有子元素的指定元素
  2. :parent
    作用: 找到有文本内容或有子元素的指定元素
  3. :contains(text)
    作用: 找到包含指定文本内容的指定元素
  4. :has(selector)
    作用: 找到包含指定子元素的指定元素
示例代码：
<script>
        var $div = $("div:empty");
        console.log($div);

        var $div = $("div:parent");
        console.log($div);

        var $div = $("div:contains('我是div')");
        console.log($div);

        var $div = $("div:has('span')");
        console.log($div);
</script>



<body>

    <div></div>
    <div>我是div</div>
    <div>他们我是div123</div>
    <div><span></span></div>
    <div><p></p></div>

</body>
