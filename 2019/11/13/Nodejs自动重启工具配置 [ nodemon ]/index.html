Nodejs自动重启工具配置 [ nodemon ]

Nov 13, 2019点击

1. 安装依赖包

2.修改package.json文件

找到scripts
"scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
在scripts下面添加start
"scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "start": "nodemon app.js"
  },

3. app.js根目录下打开终端，键入npm run start

4. 修改后，ctrl+s保存之后，项目变化重新启动。
